# ClassGuard – Student Risk Prediction

## 📌 Project Overview
ClassGuard is a machine learning application designed to predict whether a student is **Safe ✅** or **At Risk ⚠️** based on their **attendance percentage** and **marks/total score**.  
The goal is to help educators identify at-risk students early and provide timely support.

---

## 🎯 Features
- Input student details (Name, Branch/Section, ID No)
- Enter **Attendance Percentage** and **Marks**
- Predict risk status using **Logistic Regression**
- Output with clear taglines:
  - Safe → *“On track for success.”*
  - At Risk → *“Needs immediate attention.”*

---

## 🗂 Dataset
- **Fields:**
  - Attendance Percentage (0–100)
  - Marks / Total Score (0–100)
- **Target Output:**
  - Safe (1)
  - At Risk (0)

---

## ⚙️ Model
- Algorithm: **Logistic Regression**
- Accuracy: ~**99.9%**
- Training/Testing split applied for evaluation
